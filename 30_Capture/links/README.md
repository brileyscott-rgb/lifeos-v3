# Links

## What goes here

URLs with optional notes, from `/link` commands or similar.

## What should not go here

Notes without URLs, ideas, project updates.

## Filename format

`YYYYMMDD_HHMMSS_source_link_short-description.md`

## Content format

```markdown
# Link
URL: <url>
Note: <optional note>
Source: <telegram|manual|web|cli>
Captured at: <timestamp>
```
