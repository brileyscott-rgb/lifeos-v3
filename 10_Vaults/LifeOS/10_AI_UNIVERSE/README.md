# AI Universe

## Purpose

Govern LifeOS V3 AI agents, permissions, schemas, logs, prompts, runtime context, and system boundaries.

## Core Principles

- Agent behavior is permission-based.
- Meaningful actions are logged.
- AI Mirror observes only.
- Advanced infrastructure is allowed when governed, observable, and recoverable.
- Direct mutation requires role permission, path permission, approval-tier permission, and event logging.
